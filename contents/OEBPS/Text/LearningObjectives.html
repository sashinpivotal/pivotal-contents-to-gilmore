Introduction

This document is designed to capture the overall learning objectives for the Core Spring course. It was created after the course itself was written so, while it mirrors the current set of topics, it isn’t 100% aligned with the current state of the course material itself.

The objectives are written based on the information at the Objectives Builder site.

Course Objectives

At the end of the class the student will be able to:

  • Use Spring and Spring Boot to build Enterprise Applications

  • Describe why you would use Spring and the Dependency Injection Pattern

  • Configure Spring using Java Configuration and/or Annotations (@Component, @Autowired) and component-scanning

  • Describe the internal life-cycle of Spring bean

  • Explain the purpose of a extension points, such as BeanFactoryPostProcessor and BeanPostProcessor - and the difference between them

  • Explain the Separation of Concerns concept

  • Implement and test an Aspect

  • Configure and use JdbcTemplate and JPA in with Spring

  • Describe the reasons for using Transactions and how to configure them using Spring AOP

  • Write a simple Controller using a Model and View

  • Build an application using Spring Boot

  • Describe Spring Boot concepts such as opinionated configuration and starter dependencies

  • Build a RESTful server using Spring MVC

  • Leverage Spring Boot's Actuators and Health Indicators

  • Use Spring Security to secure Web and REST endpoints

Module Objectives

For each module, the second slide (after the Title slide) should be entitled "Overview" with the first line: "After completing this lesson, you should be able to".

For each of the modules below, students will be able to:

  1. Overview of the Spring Framework

    • Define the Spring Framework

    • Explain what Spring is used for

    • Discuss why Spring is successful

    • Explain where it fits in your world

  2. Dependency Injection Using Java Configuration - 1

    • Define Spring Beans using Java code

    • Access Beans in the Application Context

    • Handle multiple Configuration files

    • Handle Dependencies between Beans

    • Explain and define Bean Scopes

  3. Dependency Injection Using Java Configuration - 2

    • Use external Properties to control Configuration

    • Demonstrate the purpose of Profiles

    • Use the Spring Expression Language (SpEL)

    • Explain how Spring Java Configuration implements Singleton Beans

  4. Annotation-Driven Configuration

    • Explain and use Annotation-based Configuration

    • Discuss Best Practices for Configuration choices

    • Use @PostConstruct and @PreDestroy

    • Explain and use Spring’s “Stereotype” Annotations

  5. The Bean Lifecycle

    • Explain the 3 Phases of the Spring Bean Lifecycle

    • Use a BeanFactoryPostProcessor and a Bean PostProcessor

    • Explain how Spring Proxies add behavior at runtime

    • Explain how pre-destroy is implemented

  6. Developing Aspects with Spring AOP

    • Explain the concepts behind AOP and the problem that it solves

    • Implement and deploy an Advice using Spring

    • Use AOP Pointcut Expressions

    • Explain the 5 different Types of Advice and when to use them

  7. Introduction to Testing with Spring

    • Explain the concepts of TDD

    • Write tests using JUnit

    • Write Integration tests using Spring

    • Configure tests using Spring Profiles

    • Extend Spring Tests to work with Databases

  8. Introduction to Spring JDBC

    • Explain the problems with traditional JDBC

    • Use and configure Spring’s JdbcTemplate

    • Execute queries using callbacks to handle result sets

    • Handle Exceptions

  9. Transaction Management with Spring

    • Explain why Transactions are used

    • Describe and use Spring Transaction Management

    • Configure Isolation Levels and Transaction Propagation

    • Setup Rollback rules

    • Use Transactions in Tests

  10. Introduction to Spring Boot

    • Explain what Spring Boot is and why it is Opinionated

    • Create a Spring Boot project from a starter

    • Determine what starter components to package

    • Add additional starters to package

  11. Leveraging Autoconfiguration

    • Demonstrate how Spring Boot leverages auto-configuration

    • Utilize auto-configuration to simplify JDBC project configuration and initialization

    • Determine behavior of various configuration elements such as @SpringBootApplication

    • Determine what configuration has been detected and implemented by Spring Boot at startup

  12. Spring Data JPA

    • Understand the basic pattern for creating backing stores

    • Configure JPA Entity objects

    • Implement JPA using Spring Boot

    • Create Spring Data Repositories for JPA

  13. Spring MVC with Spring Boot

    • Describe the general Spring MVC architecture

    • Create simple RESTful controller to handle a 'GET' request

    • Configure packaging to create a fat JAR

    • Run a Web application with an embedded server

  14. Creating RESTful Applications

    • Create REST endpoints for various verbs (GET, PUT, POST, DELETE)

    • Process arguments from the URL

    • Utilize message converters to return data as JSON or XML

    • Utilize RestTemplate to invoke services

  15. Spring Security

    • Explain basic Security Concepts

    • Setup Spring Security in a Web Environment

    • Configuring Web Authentication

    • Use Spring Security’s Tag Libraries

    • Setup Method Security

  16. Spring Boot Actuators and Health Indicators

    • Enable Spring Boot Actuators

    • Configure what actuator endpoints get enabled

    • Extend health endpoint to add custom health indicators

  17. Testing with Spring Boot

    • Enable Spring Boot testing

    • Utilize mocks to simplify testing

    • Enable slices using auto configured labs
